# LittleLend Backend Project

### Installation

```
git clone https://github.com/SaraOsdi/LittleLend.git
cd LittleLend
npm i 
npm start 
```

### Enviroment

```
PORT = port
JWT_SECRET= 
JWT_TIME= 
DB = "url"
AWS_ACCESS_KEY_ID = 'key'
AWS_SECRET_ACCESS_KEY = 'key'
AWS_REGION = 'region'
BUCKET_NAME = ''
```
